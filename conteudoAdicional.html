Conteúdo adicional

Você aí, sim você mesmo, ficou com vontade de fazer o seu robô? nesta página serão citados alguns tutoriais e dicas. É importante saber que, existem vários jeitos de se fazer um, tais como: virtuais com machine learning ou deep learning, várias linguagens de programação disponíveis, lego, material pronto como os usados aqui e entre outros. Vamos resumir aqui tais tópicos mas é aconselhável que vocês pesquisem para se aprofundar no assunto.

Tipos de robôs

Maneiras de fazer seu proprio robô.

Machine learning

Ou aprendizado de máquina, é algo que vem ganhando atenção nos últimos anos, se baseia em treinar um "cérebro vazio" usando dados coletados, geralmente feito em python.

Deep learning

Ou as famosas "IA’s", um assunto que "explodiu" dado a quantia de chats sendo criados, se baseia em treinar uma rede neural artificial, geralmente feito em python também, porem é possível fazer em C#, Java e entre outras.

Recomendação de linguagem

Como citado a esquerda, várias linguagens de programação fazem o mesmo serviço, nos aconselhamos que você procure por si só e escolha a que mais se encaixa no seu gosto.

Robôs "físicos"

Não dá para falar de robôs sem citar Arduino, o Arduino serve para facilitar o aprendizado de programação usando robôs, tendo inclusive kits de Arduino que são recomendados.

Com arduino

Arduino vem com peças prontas, você monta e programa, dado a facilidade junto a quantia de tutoriais na internet, é recomendado para iniciantes.

Com lego

Existem pecas especificas de lego, inclusive com uso de pneumática, para fazer robôs, não é recomendado para iniciantes.
